**[Zimara – Local security checks before you push](https://github.com/oob-skulden/zimara)**  
  Catches secrets, private keys, risky files, and accidental exposures in your repo *before* they hit GitHub or production.

**[APIpreFlight.dev – Pre-integration security clearance for vendor APIs](https://github.com/oob-skulden/apipreflight.dev)**

  → [Download script](https://github.com/oob-skulden/apipreflight.dev/blob/main/apipreflight_v0_15_2.py)
